# Heart Disease Prediction using Machine Learning

This project predicts whether a patient has heart disease using machine learning models based on clinical features such as age, cholesterol, chest pain type, and maximum heart rate.

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- ROC-AUC

## Dataset
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset
